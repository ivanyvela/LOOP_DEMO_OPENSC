# LOOP_DEMO_OPENSC: Redox Probe Open Science Platform

This repository hosts an open-science web application for visualizing redox interfaces in groundwater.

## Project Structure
- `web-app/`: The React application.
- `Python_Redox_Geochemistry/`: Raw data, metadata, and scientific papers.
- `Borehole_Multiplots/`: Animation logic for boreholes.
- `Nitrate_Focus/`: Statistical logic for nitrate reduction analysis.
- `Interface_Detection/`: Algorithmic interface detection notebooks.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Development
1. Navigate to the web app:
   ```bash
   cd web-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```

## Deployment
The project is configured for GitHub Pages.
```bash
npm run deploy
```
